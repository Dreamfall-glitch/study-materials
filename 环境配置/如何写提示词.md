# 模版

假设你是 **填入一个具体的角色**。

我目前 **描述你具体的身份、处境和关键细节，例如：一个刚起步的护肤博主，粉丝500人，内容主要是产品实拍但点赞很少**。

我想 **明确的、可衡量的目标，例如：在3个月内将粉丝涨到1万，并打造出「成分党」的专业人设 

我目前有 **列出你的资源和限制，例如：只有一部手机拍照，没有专业灯光，每天只有1小时创作时间，预算为2000元**。

请帮我 **用具体的动词描述你的请求，例如：策划3个能引发争议的爆款选题 / 设计一套完整的专业知识内容框架 / 写一篇口语化的视频脚本**。

以 **选择你想要的格式，例如：分段阐述，按数字序号排列 / 先用一个表格总览，再分段详细阐述 / 提供可直接复制粘贴的清单体**回答。

用 **描述你想要的风格，例如：像朋友聊八卦一样有感染力 / 专业冷静、多用数据支撑 / 犀利毒舌但一针见血**的口吻和语气。
